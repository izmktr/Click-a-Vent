# アイコンファイルについて

このフォルダには、Chrome拡張機能用のアイコン画像を配置する必要があります。

## 必要なファイル

- `icon16.png` (16x16ピクセル)
- `icon48.png` (48x48ピクセル)  
- `icon128.png` (128x128ピクセル)

## アイコンの作成方法

1. **SVGからPNGへの変換**
   - `icon128.svg` ファイルが含まれています
   - オンライン変換ツールを使用してPNGに変換:
     - https://cloudconvert.com/svg-to-png
     - https://svgtopng.com/
     - https://www.aconvert.com/image/svg-to-png/
   
2. **サイズ別の作成**
   - 128x128pxで変換した後、画像編集ツールで16x16と48x48にリサイズ
   - または、変換時に各サイズで個別に変換

3. **独自アイコンの使用**
   - カレンダーとクリックカーソルをモチーフにしたデザインを推奨
   - 背景色: #1a73e8 (Googleブルー)
   - PNG形式、透過背景も可

## 一時的な対応

開発・テスト時は、シンプルな単色のPNGファイルで代替可能です。
オンラインツールで簡単にダミーアイコンを作成できます。
